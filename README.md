# DevVerse 🚀

DevVerse is a modern developer productivity dashboard designed to help programmers stay organized, focused, and productive. It combines essential tools into a single, clean interface, making it easier to manage coding sessions, tasks, notes, and GitHub activity.

## ✨ Features

### 📊 GitHub Profile Analytics

* Search any GitHub user
* View profile information
* Public repositories count
* Followers and following statistics
* Profile avatar and bio

### 🍅 Pomodoro Timer

* 25-minute focus sessions
* 5-minute break sessions
* Start, pause, and reset controls
* Productivity-focused workflow

### 📝 Smart Notes

* Create and manage notes
* Edit existing notes
* Delete notes
* Local storage support

### ✅ Task Manager

* Add tasks
* Mark tasks as completed
* Delete tasks
* Persistent storage using Local Storage

### 🌙 Modern UI

* Dark/Light Mode
* Responsive Design
* Smooth Animations
* Mobile Friendly Layout

### 📅 Productivity Widgets

* Current Date & Time
* Daily Motivation Quotes
* Quick Productivity Overview

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* GitHub REST API
* Local Storage API

## 🎯 Project Goals

The goal of DevVerse is to provide developers with a centralized productivity hub where they can manage their coding workflow, track tasks, take notes, and monitor GitHub activity without switching between multiple applications.
